8-Bit Buster Action (Rockman Style Game)

懐かしの8bitアクションゲーム風のシングルページWebゲームです。
HTML5 CanvasとバニラJavaScriptのみで動作し、ライブラリは一切使用していません。

🎮 概要

プレイヤーを操作してステージを進み、最奥に待ち受けるボスを倒せばクリアです。
PCブラウザおよびスマートフォン（タッチ操作）に対応しています。

✨ 特徴

レトロな8bitグラフィック: ドット絵風のビジュアルとエフェクト。

多彩なアクション:

ジャンプ & ショット（バスター）

スライディング（狭い通路を通過可能）

本格的なボス戦:

ボス部屋へのシャッター演出

ボス戦中のカメラ固定（逃げ場なし！）

専用のHPゲージ（P: プレイヤー, B: ボス）

スマホ完全対応: 画面上のバーチャルパッド（十字キー + A/Bボタン）で快適にプレイ可能。

🕹️ 操作方法

PC (キーボード)

アクション

キー操作

移動

← →

ジャンプ

Z または Space

ショット

X または S

スライディング

↓ + ジャンプ

スマートフォン (タッチ操作)

画面下部に表示されるコントローラーを使用します。

十字キー: 移動

Aボタン: ジャンプ / 決定

Bボタン: ショット

🚀 遊び方 / インストール

特別なインストールは不要です。ブラウザがあればすぐに遊べます。

このリポジトリをクローンまたはダウンロードします。

index.html (または rockman_clone_v4.html) をブラウザ（Chrome, Safariなど）で開くだけで起動します。

開発・修正

コードは単一のHTMLファイル (index.html) に全て含まれています。
テキストエディタで開けば、マップの配置やパラメータ（重力、スピードなど）を簡単に調整できます。

// 例: 定数の変更
const JUMP_FORCE = -9.5; // ジャンプ力の調整
const SPEED = 3;         // 移動スピード
